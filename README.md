# Local Legal Contract Semantic Search Backend

这是一个**本地化、高精度、跨语言**的英文法律合同中文语义检索后端项目。旨在通过中文意图精准定位英文条款，确保数据隐私与检索准确性。

## 🚀 快速开始 (Quick Start)

1. **环境准备**:
    * 确保安装 Python 3.11+
    * 安装依赖: `pip install -r requirements.txt`

2. **运行**:
    * 请参考 `api_interface.py` 中的示例代码或通过 `main_ui_qt.py` (如果有) 启动应用。
